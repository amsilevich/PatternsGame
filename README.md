# Правила сборки
- Клонировать репозиторий
- Предоставить права доступа сборочнму скрипту
- Запустить сборочный скрипт
- Запустить исполняемый файл
    ```sh
    $ git clone https://github.com/amsilevich/PatternsGame/
    $ chmod 777 ./run.sh
    $ ./run.sh
    $ build/game
    ```
